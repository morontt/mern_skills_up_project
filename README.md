#Skills-up MERN stack application.

### Implemented functionality
1) OAuth2 
2) Post CRUD 
3) Test coverage
4) Fixtures

### Run application

Run frontend
```
cd /prject/root/folder/frontend
npm run dev
```

Run server
```
cd /prject/root/folder/server
npm run dev
```

Run server Mocha tests
```
cd /prject/root/folder/server
make run-all-tests
```

Run frontend Cypress tests
```
cd /prject/root/folder/frontend
npm run cypress:open
```
